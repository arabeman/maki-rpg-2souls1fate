# Two Souls One Fate

![Thumbnail](thumbnail.png)

**Two Souls, One Fate** is a story-driven 2D pixel RPG where every choice shapes the journey. Explore dangerous lands, battle powerful enemies, and face impossible decisions as fate forces you to choose between the people you care about most.

Built with the [Maki framework](https://www.npmjs.com/package/@tialops/maki?utm_source=chatgpt.com) using JavaScript and pixel art inspired by classic RPG adventures.

## Features

* Story-focused gameplay with meaningful choices
* Pixel art world with exploration and combat
* Multiple paths and consequences
* Boss fights and emotional moments

## Assets

Some game assets were created using resources from [Kenney Assets](https://kenney.nl/?utm_source=chatgpt.com).

## Setup

```bash
npm install --global yarn
yarn
```

## Start the Game

```bash
yarn dev
```

## Commands

* `yarn tilemap` — Open the tilemap editor
